Welcome to DotaSight

Given the server_log, Dota-Sight will populate your team and the enemies' data. You can copy and paste the entire server_log.txt in the DotA 2 directory, or just the last couple lines.

Alternatively to automate this process, download Autohotkey from www.autohotkey.com, install it, and download this script Dota-Sight script to your desktop or something, and double click on it.
If you right click on it, and edit script, you can change the file path of server_logs.txt or to simply see the source code.
When it detects that you joined a game, it will open a Dota-Sight window

Default server_log.txt file paths:
Windows C:\Program Files (x86)\Steam\steamapps\common\dota 2 beta\dota\
MAC /Library/Application Support/Steam/steamapps/common/dota 2 beta/dota
Linux ~/.steam/root/SteamApps/common/dota 2 beta/dota
